Fazendo citações/referências

<blockquote>

O Elemento HTML <blockquote> (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. A URL para a fonte da citação pode ser dada usando o atributo cite.

<cite>

De acordo com MDN Web Docs - cite:
O elemento HTML <cite> representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência, que pode ser em uma forma abreviada de acordo com as convenções usadas para a adição dos metadados de citação.

<q>

O elemento HTML <q> indica que o texto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo ou quebras de linha. - MDN Web Docs - q.
